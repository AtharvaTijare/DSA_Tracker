# Tracker

### 1. Check/Install Node.js
React needs a "motor" to run on your laptop. Node.js is that motor.
Command: " node -v " . 
 What it does: Checks if Node.js is installed. If not, you must download it from nodejs.org.

### 2. Navigate to Project
You must be inside the specific folder where the project files live.
Command: " cd folder-name " . 
 What it does: Moves your terminal's focus into your project directory.

### 3. Install Dependencies
React projects don't come with all their libraries attached (to save space). You have to download them.
Command: " npm install ". 
 What it does: Reads the package.json file and downloads every library your project needs into a folder called node_modules.

### 4. Start the Application
Once the libraries are downloaded, you can actually turn the app "on."
Command: " npm start ".(for Create-React-App). 
 What it does: Compiles the code and starts a local web server on your laptop.

5. Open in Browser
The terminal will give you a link, usually localhost:5173 or 3000.

Action: Copy the link into Chrome/Edge.

What it does: Connects your browser to the local server so you can see and interact with your app.
